Kaplan International Berkeley Library System

Library Management System built for the Berkeley branch of Kaplan International Colleges.

  • Increased library budget by ~15% by eliminating loss through designing library circulation management tool with Ruby on Rails for language school to organize their physical library used by 200+ students and 10 staff.
  • Designed streamlined checkout system UI/UX using JavaScript, jQuery, AJAX, and PureCSS which reduced book checkout transaction time by ~75%.
  • Engineered PostgreSQL DB schema using ActiveRecord to store 1,200+ books with the features to import and export data from pre-existing databases and create custom reports (CSV and PDF).
  • Collected and incorporated feedback from 30% of userbase to refactor features to improve UX and workflow.
  • Achieved 100% coverage of codebase by following TDD and writing robust test suite with RSpec and Capybara.

Originally built as my contribution to a larger e-commerce web app, Resume Builder is a demonstration of Rails' nested model forms. It can build a resume object with an arbitrary number of associated jobs, education experiences, skills, etc... and dynamically generate forms to accommodate them.

Codex

CODEX is a self-study tool that encourages you to read more code. The platform enables you to create scheduled "servings" of high quality code direct from Github based on the language and topic of your choice. Timely notifications are delivered via SMS message or email utilizing the Twilio and SendGrid APIs. Every week, Codex will send you new repositories to examine. Repositories are selected based on their popularity, number of contributors, and frequency of activity. It was built in five days as the graduation project for Dev Bootcamp by a team of four student developers. I was primarily responsible for implementing the Rails back-end, and I contributed a significant amount of jQuery to the front-end as well.

StudyCard

StudyCard is a flashcard-based memorization tool based on the SuperMemo SM-2 algorithm. Cards will be served up for review to maximize learning efficiency and simplify the organization and review of large flashcard collections. It was built using Rails in the back and JQuery in the front.
