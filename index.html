Kaplan International Berkeley Library System

This app was built for the Berkeley branch of Kaplan International College to replace their paper card based system. Simple enough to use entirely on a mobile device, it allows teachers to check books out to students, check the books back in, manage books and students, and generate reports. It was built using Ruby on Rails and JavaScript.

StudyCard

StudyCard is a flashcard-based memorization tool based on the SuperMemo SM-2 algorithm. Cards will be served up for review to maximize learning efficiency and simplify the organization and review of large flashcard collections. It was built using Rails in the back and JQuery in the front.

Codex

CODEX is a self-study tool that encourages you to read more code. The platform enables you to create scheduled "servings" of high quality code direct from Github based on the language and topic of your choice. Timely notifications are delivered via SMS message or email utilizing the Twilio and SendGrid APIs. Every week, Codex will send you new repositories to examine. Repositories are selected based on their popularity, number of contributors, and frequency of activity.
