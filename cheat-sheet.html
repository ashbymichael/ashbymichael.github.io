Keyboard Shortcuts

Atom

  • Toggle Comment: ⌘ + /
  • Toggle File Finder: ⌘ + p
  • Go to line: ctrl + g
  • Open Pane: ⌘ + k, then an arrow key
  • Indent/Outdent: ⌘ + ]/[
  • move line up/down: ctrl + ⌘ + up/down
  • Select line: ⌘ + l
  • Select word: ⌘ + d
  • Select to end/beginning of word: opt + shift + right/left
